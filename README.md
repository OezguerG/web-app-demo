# Web Engineering 2 – Full-Stack Web Application

A full-stack web application developed for the Web Engineering 2 course. This project demonstrates modern web development practices, including user authentication, REST APIs, role-based access control, and CRUD operations for managing areas and themes.

## Features

- User authentication with JWT tokens
- REST API endpoints for backend interaction
- Full CRUD operations for areas and themes management
- Role-based access control (Admin and regular users)
- Responsive frontend design

## Technologies

- React
- Bootstrap
- Express.js
- MongoDB
- JWT (JSON Web Tokens)
