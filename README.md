# Web Engineering 2 – Full-Stack Web Application

[Deutsche Version](#deutsche-version) | [English Version](#english-version)

## Deutsche Version

### 🎯 Projektübersicht

Eine Full-Stack Webanwendung, entwickelt für den Kurs **Web Engineering 2**. Das Projekt demonstriert moderne Webentwicklungsmethoden, einschließlich Authentifizierung, REST APIs und rollenbasierter Zugriffssteuerung.

### ✨ Implementierte Features

• **Benutzer-Authentifizierung:** Sicheres Login/Logout mit JWT Tokens
• **REST API:** Endpunkte zur Interaktion mit dem Backend
• **Gebiete & Themen Verwaltung:** Vollständige CRUD-Operationen (Erstellen, Lesen, Aktualisieren, Löschen)
• **Benutzerrollen:** Admin und normale Benutzer mit unterschiedlichen Berechtigungen
• **Frontend:** React und Bootstrap für responsive Design
• **Backend:** MongoDB und Express zur Datenverwaltung
• **Rollenbasierte Zugriffskontrolle:** Unterschiedliche Funktionen für Admin vs. normale Benutzer

### 🛠️ Technischer Stack

• **Frontend:** React, Bootstrap
• **Backend:** Express.js
• **Datenbank:** MongoDB
• **Authentifizierung:** JWT (JSON Web Tokens)
• **APIs:** RESTful API Design

## English Version

### 🎯 Project Overview

A full-stack web application developed for the **Web Engineering 2** course. This project demonstrates modern web development practices, including authentication, REST APIs, and role-based access control.

### ✨ Features Implemented

• **User Authentication:** Secure login/logout using JWT tokens
• **REST API:** Endpoints for backend interaction
• **Areas & Themes Management:** Full CRUD operations (Create, Read, Update, Delete)
• **User Roles:** Admin and regular users with different permissions
• **Frontend:** React and Bootstrap for responsive design
• **Backend:** MongoDB and Express for data management
• **Role-Based Access Control:** Different functionalities for admin vs. regular users

### 🛠️ Technical Stack

• **Frontend:** React, Bootstrap
• **Backend:** Express.js
• **Database:** MongoDB
• **Authentication:** JWT (JSON Web Tokens)
• **APIs:** RESTful API Design
